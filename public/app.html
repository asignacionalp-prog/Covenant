Migration & Bulk Import
For first-time setup. Already had partners working before adopting Covenant? Use these tools to import historical attendance and client payments from spreadsheets — no need to re-key everything by hand.
Step 1 — Time entries (attendance)

Required columns: partner_email, date (YYYY-MM-DD), hours_worked. Optional: note.
One row per partner per day. Fractional hours allowed (e.g. 7.5).

Step 2 — Client payments

Required columns: client_name, date (YYYY-MM-DD), amount, currency (PHP/USD/etc), period (YYYY-MM).
Optional: fx_rate (defaults to typical rate for non-PHP), note.

Tip: Make sure partners and clients exist in Covenant first (with matching emails / names). The import will skip rows that reference unknown partners or clients and report what was skipped.
